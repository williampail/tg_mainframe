# Telegram Post Rewriter Bot

Бот для Telegram, который принимает посты, переписывает их в вашем стиле и публикует в ваш канал.

## Возможности

- ✅ Приём постов через пересылку сообщений
- ✅ Переписывание текста с помощью AI (OpenAI GPT)
- ✅ Публикация в ваш Telegram канал
- ✅ Поддержка текста и фото
- ✅ Настраиваемый стиль письма

## Требования

- Python 3.9+
- Telegram Bot Token
- OpenRouter API Key (https://openrouter.ai/keys)
- Ваш канал в Telegram

## Установка

### 1. Клонируйте или создайте файлы проекта

Убедитесь, что все файлы находятся в одной папке:
```
tg_mainframe/
├── bot.py
├── config.py
├── style_rewriter.py
├── channel_publisher.py
├── requirements.txt
├── .env.example
└── .gitignore
```

### 2. Создайте виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

Скопируйте `.env.example` в `.env`:
```bash
cp .env.example .env
```

Отредактируйте `.env` и заполните своими данными:

```env
# Telegram Bot Token (получите у @BotFather)
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

# ID вашего канала (например, @my_channel или -1001234567890)
CHANNEL_ID=@my_channel

# OpenRouter API Key (получите на https://openrouter.ai/keys)
OPENROUTER_API_KEY=sk-or-xxxxxxxxxxxxx

# Модель (бесплатные: google/gemma-2-9b-it:free, meta-llama/llama-3-8b-instruct:free)
MODEL=google/gemma-2-9b-it:free

# Описание вашего стиля письма
WRITING_STYLE=Пишите в дружеском, вовлекающем тоне. Используйте ясный и лаконичный язык. Добавляйте личные инсайты и делайте сложные темы доступными.
```

## Получение Telegram Bot Token

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям (введите имя и username бота)
4. Скопируйте полученный токен

## Получение OpenRouter API Key

1. Зарегистрируйтесь на [openrouter.ai](https://openrouter.ai/)
2. Перейдите в раздел Keys
3. Создайте новый ключ
4. Используйте бесплатные модели: `google/gemma-2-9b-it:free`, `meta-llama/llama-3-8b-instruct:free`

## Настройка канала

1. Создайте канал в Telegram (если ещё не создан)
2. Добавьте вашего бота в канал как **администратора** с правами на публикацию сообщений
3. Узнайте ID канала:
   - Для публичного канала: используйте `@username_канала`
   - Для приватного: перешлите сообщение из канала в [@GetMyIDBot](https://t.me/GetMyIDBot)

## Запуск бота

```bash
python bot.py
```

## Использование

1. Отправь боту команду `/start` для проверки
2. **Перешли** пост из другого канала боту
3. Бот подтвердит получение
4. Отправь команду `/rewrite`
5. Бот перепишет пост в стиле MAINFRAME
6. Бот опубликует результат в твой канал

### Команды:

| Команда | Описание |
|---------|----------|
| `/start` | Показать справку |
| `/status` | Проверить статус бота |
| `/rewrite` | Переписать и опубликовать последний пересланный пост |

## Настройка стиля письма

Откройте `.env` и измените параметр `WRITING_STYLE`:

Примеры:

**Деловой стиль:**
```env
WRITING_STYLE=Пишите в профессиональном, деловом стиле. Используйте точные формулировки, избегайте эмоциональных выражений. Структурируйте информацию логично.
```

**Неформальный стиль:**
```env
WRITING_STYLE=Пишите неформально, как будто общаетесь с другом. Используйте эмодзи, разговорные выражения, юмор когда уместно.
```

**Минималистичный стиль:**
```env
WRITING_STYLE=Пишите кратко и по делу. Удаляйте лишние слова. Каждое предложение должно нести ценность.
```

## Troubleshooting

**Бот не публикует в канал:**
- Убедитесь, что бот добавлен в канал как администратор
- Проверьте правильность CHANNEL_ID

**Ошибка OpenRouter:**
- Проверьте OPENROUTER_API_KEY
- Убедитесь, что модель доступна (проверьте на openrouter.ai/models)

**Бот не отвечает:**
- Проверьте TELEGRAM_BOT_TOKEN
- Убедитесь, что бот запущен (`python bot.py`)
